NuRoute

Running is a great way to exercise. It is arguably the most popular way to exercise and can also be monotonous and repetitive. Not to worry! NuRoute is here to make running more exciting by randomizing your running route. A new route allows for a more enjoyable running experience because it allows for new scenery and running grades. Not to mention this takes the thought out of "where would I like to run today?" This app guarantees that you wont run the same route the next time, or the time after that! What are you waiting for? Get out there and run with NuRoute!

NuRoute: Lets Run!

To begin your run click on the 'Continue to the Map' button. You will be redirected to the running map and will be prompted to provide the starting location of your run. This location will require an address. The distance field will be filled with the desired length of the run.
